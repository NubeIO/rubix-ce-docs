## Course 3: Wires — Wiresheet, Nodes & Flow Logic
Suggested modules to cover:

- Getting Started with Wires — overview of the Wires environment
- Wiresheet — creating and managing wire-based programs for controlling and monitoring devices 
- Sub-flows — using folders and sub-flows to organise logic
- Export and Import — sharing and reusing flow programs
- Point Mapping — mapping data points within the wiresheet
- Payload — using the Set Payload function to send values to the Wires runtime without redeploying the flow
